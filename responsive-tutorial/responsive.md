# レスポンシブ・チュートリアル
HTMLだけでレスポンシブを実装します。

## ブレイクポイント
Bootstrapには便利なブレイクポイントの機能があります。これを使うことでHTMLだけでレスポンシブなページを構築することが可能です。


## グリッドシステムのレスポンシブ

ブレイクポイントを使えば、グリッドの行と列を自由に変化させることが可能です。

タブレットサイズで1列にする例

```html
<div class="row">
  <div class="col-md">col-md</div>
  <div class="col-md">col-md</div>
</div>
```
